# 國小國語課程內容產生器

## 目的
協助家長和老師快速建立國小國語課程的學習內容，包含生字表、注音對照、造詞、例句等。

## 使用情境

- 使用者需要根據課本內容建立字詞庫
- 使用者想要產生某個年級的常用字表
- 使用者需要為特定課文分析生字和詞語
- 使用者想要產生練習題目

## 工作流程

### 1. 確認需求
詢問以下資訊：
- 年級（一到六年級）
- 課本版本（南一、康軒、翰林）
- 學期（上學期/下學期）
- 課次

### 2. 產生內容格式

每個字詞應包含：
```
字/詞: 學校
注音: ㄒㄩㄝˊ ㄒㄧㄠˋ
部首: 子
筆畫: 16
造詞: 學生、學習、學問
例句: 我每天都去學校上課。
```

### 3. 輸出格式

產生 JSON 格式的字詞資料，可直接匯入國小國字助手 App：
```json
{
  "lessonName": "第一課 大家的乘車公約",
  "words": [
    { "text": "約", "zhuyin": "ㄩㄝ" },
    { "text": "乘", "zhuyin": "ㄔㄥˊ" }
  ]
}
```

### 4. 注音產生規則

- 使用臺灣標準注音符號（ㄅㄆㄇ系統）
- 聲調標記：一聲不標、二聲ˊ、三聲ˇ、四聲ˋ、輕聲˙
- 多音字標注最常用讀音，並附註其他讀音
- 破音字需根據詞語上下文決定讀音

### 5. 年級對照參考

- 一年級：注音符號、基礎筆畫、約300字
- 二年級：常用字約600字、簡單造詞
- 三年級：常用字約1000字、成語入門
- 四年級：常用字約1500字、修辭初步
- 五年級：常用字約2000字、閱讀理解
- 六年級：常用字約2500字、作文訓練

## 注意事項

- 所有內容需使用正體中文（繁體中文）
- 注音使用臺灣教育部頒佈的標準
- 避免使用簡體字或大陸用語
- 考慮兒童的認知發展階段，由簡入繁
